# **App Name**: VideoReview

## Core Features:

- Video Upload: Video Upload: Allows users to upload video files directly through a drag-and-drop interface or by selecting files from their device.
- Link Generation: Link Generation & Sharing: Generates a unique, shareable link upon successful video upload, enabling easy distribution for review.
- Timestamped Comments: Timestamped Comments: Allows viewers to leave comments at specific timestamps within the video, facilitating precise feedback.
- Comment Checkmarks: Comment Checkmark System: Allows users to mark comments as 'done', persisting the state in localStorage for a personalized review experience.
- AI Comment Tool: AI-Powered Comment Summary: Uses a tool to generate a summarized list of the key feedback points from the comments section, to give an overview to the video creator

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a modern and sophisticated feel.
- Background color: Dark gray (#303030) to create a dark mode aesthetic.
- Accent color: Electric purple (#8C30F5) for interactive elements and highlights.
- Headline font: 'Poppins' (sans-serif) for headings, providing a modern and geometric look.
- Body font: 'PT Sans' (sans-serif) for body text, ensuring readability and a clean aesthetic.
- Use minimalistic, outlined icons for interactive elements.
- A clear division between the video and comments sections, with video taking up approximately 65% of the width and comments 33%.
- Subtle animations for interactive feedback like when submitting or approving comments
